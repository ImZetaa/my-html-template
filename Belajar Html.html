Pendahuluan

HTML (Hypertext Markup Language) adalah bahasa pemrograman yang digunakan untuk membuat struktur dan konten dari halaman web. Dalam pengembangan web, HTML berperan sebagai tulang punggung yang mendefinisikan elemen-elemen dasar dan hubungan antara mereka.

Did you know? HTML telah ada sejak tahun 1990 dan merupakan salah satu teknologi inti yang membentuk World Wide Web (WWW). Awalnya, HTML hanya memiliki versi dasar, tetapi seiring waktu, telah mengalami perkembangan yang signifikan dan saat ini digunakan dalam versi HTML5 yang memiliki banyak fitur canggih dan kemampuan multimedia.

Selain itu, HTML juga merupakan bahasa yang mudah dipelajari. Dengan pengetahuan dasar HTML, Anda dapat membuat struktur dasar halaman web dan menambahkan elemen-elemen seperti teks, gambar, tautan, dan daftar. Kemudian, Anda dapat mengkombinasikan HTML dengan CSS (Cascading Style Sheets) dan JavaScript untuk menciptakan tampilan dan interaktivitas yang lebih dinamis.

Dalam tutorial ini, kita akan menjelajahi dasar-dasar HTML, termasuk sintaks dasar, tag HTML umum, penggunaan atribut, dan banyak lagi. Mari kita mulai mempelajari HTML dan membuka pintu ke dunia pengembangan web!

Syntax Dasar

Contoh elemen HTML:

                
<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Ini adalah judul halaman</h1>
    <p>Ini adalah paragraf.</p>
</body>
</html>

Tag HTML

Ini adalah daftar beberapa tag HTML yang umum digunakan:

  • <h1> - <h6>: Judul
  • <p>: Paragraf
  • <a>: Tautan
  • <img>: Gambar
  • <ul> dan <li>: Daftar tak-berurutan
  • <ol> dan <li>: Daftar berurutan
  • <table>, <tr>, <td>: Tabel

Atribut

Contoh penggunaan atribut:

                
<a href="https://www.example.com">Tautan ke contoh.com</a>
<img src="gambar.jpg" alt="Deskripsi gambar">
<table border="1">
    <tr>
        <td>Sel 1</td>
        <td>Sel 2</td>
    </tr>
</table>
